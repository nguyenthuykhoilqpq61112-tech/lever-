# 用户画像地图

> 使用边界：当需要识别考生主画像、能力约束、长期路径或扩展场景时读取本文档；产出画像组合、不适合策略和分流方向；不要把画像标签当成固定命运或唯一方案。

当用户画像明显不止“窄选择普通本科考生”，或需要先判断应采用哪类策略时，读取本文档。本文档用于把考生放入“录取空间 × 能力约束 × 长期路径”的三维路由中，再选择候选池和方案包。

## 使用原则

1. 普通本科志愿仍是主流程；专科/高职、复读、艺体、专项、定向等作为扩展画像处理。
2. 画像不是标签，而是决策路由。一个考生可以同时属于多个画像，例如“中上分段 + 语言表达型 + 体制考试倾向 + 低预算约束”。
3. 先判断安全边界，再判断杠杆机会。任何画像都不能跳过录取概率、专业组下限、成本和规则核验。
4. 所有画像都要提高长期发展机会，但杠杆形式不同。高分考生要避免高分低配和过热峰值买入，压线考生要审计本科执念，专科/高职考生要看技能和行业通道。
5. 当画像之间冲突时，优先级为：不可接受项和安全底线 > 学生真实能力 > 家庭成本和身心边界 > 长期路径 > 短期偏好。

## 核心受益画像

本 Skill 可以服务多种考生。最容易看到“低估机会发现”价值的是选择空间不宽、常规匹配容易吃亏的考生；但高分宽选择考生同样需要长期发展机会审计，重点是避免在热门学校、热门城市、热门专业或同层次过热选项上用高分买到低长期回报。

| 画像 | 为什么需要本 Skill | 交互提醒 |
| --- | --- | --- |
| 中位窄选择 | 有本科机会，但随大流容易进入低特色、低杠杆或竞争过热的结果 | 先扩大机会雷达，再看低估机会是否有补偿 |
| 本科压线 | 容易为了本科身份接受高费、低质或不可接受调剂 | 先审计本科质量底线和真实保底 |
| 本专科临界 | 几分差距可能改变本科/专科、职业本科或优质高职路径 | 不把本科执念当唯一答案，比较身份、成本和发展通道 |
| 尴尬分段 | 多几分可能够到高平台，少几分可能滑向明显下限 | 志愿组合要分散风险，保底必须可接受 |
| 高竞争省份或信息弱势家庭 | 容易被本省拥挤选择、热门城市、短视频和熟人经验带偏 | 先核验规则和位次，再处理偏好和情绪 |
| 有执行力但分数不宽 | 适合用学风、保研/考研、转专业、证书和平台升级窗口做后续跃迁 | 必须确认学生愿意把录取后的三四年当作继续升级期 |

高分宽选择考生不应强行套用低分段“抄底”逻辑，但必须做强学科、长期上限、科研平台、专业过热、同层次替代、入学后发展节点和高分低配审计。

## 画像路由步骤

1. 先按分数、位次、批次线和目标批次判断录取空间。
2. 再按科目强弱、学习方式、心理承受力和家庭资源判断能力约束。
3. 最后按四年后目标判断长期路径：升学、就业、体制考试、证照职业、行业系统、国际迁移、家庭资源承接或探索保留。
4. 输出画像时必须写出“不适合采用的策略”，防止只列机会、不列边界。

## 维度一：录取空间

| 画像 | 典型特征 | 核心风险 | 策略方向 |
| --- | --- | --- | --- |
| 高分宽选择 | 有较多学校、城市和专业选择权 | 高分低配、热门峰值买入、被热门城市或热门专业吸走议价能力 | 强学科、拔尖计划、强基/综评、本研贯通、科研平台、国际化资源、同层次替代、长期上限 |
| 中上分段 | 学校、城市、专业难以三全 | 主轴不清，所有条件都想要导致排序混乱 | 明确平台优先、专业优先、城市优先、升学优先或就业优先 |
| 中位窄选择 | 有本科机会但选择权不宽 | 随大流落入低杠杆学校或低特色专业 | 特殊身份、地理折价、行业特色、学风升学、可转换入口、平台升级窗口 |
| 本科压线 | 刚到或接近本科线 | 只为本科身份接受低质量、高费用或高调剂风险结果 | 公办底线、低成本、专业组下限、本科质量门槛和保底真实性 |
| 本专科临界 | 低质量本科与优质高职/职业本科同时可选 | 本科执念压倒实际发展质量 | 比较本科身份、学费、升本路径、就业通道、技能壁垒和家庭成本 |
| 专科/高职扩展 | 目标进入专科、高职或职业本科 | 被校名层级羞耻感误导，忽视行业技能 | 行业强相关、城市产业、订单班、证书、专升本、就业确定性 |
| 复读临界扩展 | 当前可填报但明显不甘心，或发挥失常 | 情绪化复读或情绪化放弃复读 | 失误原因、提分空间、心理成本、家庭支持、规则变化和当年可录取结果对比 |

## 维度二：能力约束

| 画像 | 识别信号 | 适合关注 | 需要警惕 |
| --- | --- | --- | --- |
| 理工建模型 | 数学、物理、抽象推理、编程或实验能力较强 | 工程、计算机、电子、自动化、统计、医学相关、数据方向 | 只看热门工科名称，不看课程强度和学校平台 |
| 语言表达型 | 语文、英语、政治、写作、表达、阅读理解较强 | 法学、中文、外语、新闻传播、教育、公共管理、国际事务 | 就业泛化、专业壁垒不足、只靠兴趣不建第二能力 |
| 考试执行型 | 长期备考、自律、刷题、资格证和排名竞争能力较强 | 考公考编、法考、教师资格、财会审计、考研跳板 | 把“想稳定”误读为任何专业都适合体制考试 |
| 动手技能型 | 实操、设备、实验、护理、制造、交通、服务或项目能力强 | 高职技能、工程实践、医学技术、护理、轨交、电力、制造、应急 | 只追本科名义，忽略技能壁垒和就业确定性 |
| 外语国际型 | 外语能力、跨文化兴趣、家庭预算和迁移意愿较强 | 外语、区域国别、国际组织、跨境贸易、中外合作、留学跳板 | 项目质量、认证、费用、海外升学和家庭承受力 |
| 研究潜力型 | 能长期阅读、钻研、做项目，愿意争取导师和实验室资源 | 强学科平台、保研/考研、科研训练、本研衔接 | 学校名气高但本科教学和导师资源不可达 |
| 身心边界敏感型 | 对远距离、高压、气候、身体条件、心理压力或家庭照顾依赖敏感 | 路径清晰、支持稳定、距离和强度可控的方案 | 远距离高压、转专业豪赌、超强竞争专业或孤注一掷方案 |

## 维度三：长期路径

| 路径 | 关注重点 | 核心核验 |
| --- | --- | --- |
| 升学跳板 | 学风、保研/考研、课程基础、导师资源、跨考空间 | 升学去向、专业课难度、GPA 可维护性、英语和科研机会 |
| 就业落地 | 城市产业、实习、校招、专业壁垒、校友网络 | 就业质量报告、行业岗位、实习可达性、城市成本 |
| 体制考试 | 专业代码、岗位适配、考试科目、政审体检、地区规则 | 近年招录岗位、专业限制、户籍和资格要求 |
| 证照职业 | 职业准入、证书周期、实习实践、资格考试 | 执业资格、培养资质、实习基地、失败成本 |
| 行业系统 | 行业院校、主管部门、系统认可、对口专业 | 是否仍需考试、政审、体测、面试或资格证 |
| 国际迁移 | 语言、预算、项目质量、海外升学或就业路径 | 合作项目认证、学费、升学案例、签证和家庭承受力 |
| 家庭资源承接 | 家庭行业、人脉、城市落脚点、经济缓冲 | 资源是否真实可用，是否符合学生能力和意愿 |
| 探索保留 | 目标不清晰，需要保留转向空间 | 大类培养、转专业、辅修、跨学科课程、通用能力 |

## 扩展画像

### 艺体类考生

把文化分、专业分、综合分算法、院校层级、城市资源和未来转普通就业的可能性分开判断。不要只按专业成绩或院校名气排序。

### 专项、民族、定向和公费路径

先核验资格、批次、服务年限、违约成本、政审体检、地域限制和就业义务。机会可能很大，但自由度和失败成本也可能更高。

### 军警、政法和特殊体检政审路径

提前核验身体条件、视力、体测、政审、面试、性别限制、单科要求和毕业去向。不能把提前批机会当作普通本科替代品。

### 低预算家庭考生

优先建立“成本-机会-风险”地图。高收费民办、中外合作、国际项目和远距离城市必须证明长期收益足以覆盖费用和风险。

### 信息弱势家庭考生

主动补齐规则、信源和基本概念，警惕短视频、营销号、熟人经验和招生宣传替代官方核验。

### 家长强控制或学生沉默型

报告必须标注“学生意见缺失风险”。家长可以提供资源和约束，但四年学习、生活和职业后果主要由学生承担。

### 高潜力发挥失常考生

同时评估复读、升学跳板、冷门入口、转专业和考研二次跃迁。不能只用一次考试结果否定长期潜力，也不能忽视现实录取机会。

### 职业目标过早锁死考生

把“我一定要当教师、律师、公务员、医生、程序员、金融从业者”等目标拆成课程、证书、考试、实习、岗位和失败备选。目标越单一，越要准备备选路径。

### 性别、照护和安全边界敏感场景

不要强化刻板印象，但要现实评估住宿、城市安全感、行业环境、家庭照护责任、身体条件和长期发展机会之间的冲突。

## 画像到策略路由

| 画像组合 | 优先方案 | 不应忽视 |
| --- | --- | --- |
| 高分宽选择 + 研究潜力 + 升学跳板 | 强学科、拔尖计划、科研平台、本研衔接 | 高分低配、热门专业挤压长期上限、过热方向在峰值买入 |
| 高分宽选择 + 就业落地 | 强校招平台、城市产业、专业壁垒、同层次低过热替代 | 只看城市不看学科和校招层级，或把短期热门当长期安全 |
| 中上分段 + 探索保留 | 大类培养、可转专业、平台均衡、城市资源 | 过早绑定窄专业或单一路径 |
| 中上分段 + 体制考试 | 岗位适配专业、法学/中文/财会/计算机等方向审计 | “稳定”叙事替代真实招录限制 |
| 中位窄选择 + 升学跳板 | 学风升学、地理折价、可转换入口 | 把热门城市普通平台误判为更优 |
| 中位窄选择 + 就业落地 | 城市产业、实习可达、专业壁垒和家庭城市资源 | 把保研考研氛围误写成个人必然收益 |
| 中位窄选择 + 行业系统 | 特殊身份、行业特色、系统认可 | 行业通道是否仍需考试和资格 |
| 本科压线 + 低预算 | 公办底线、低成本城市、真实可接受保底 | 高费低质本科和不可接受调剂 |
| 本专科临界 + 动手技能 | 优质高职、职业本科、订单班、专升本 | 本科身份执念压倒技能和就业 |
| 专科/高职扩展 + 就业落地 | 城市产业、证书、实训、校企合作 | 只看校名，不看具体就业通道 |
| 复读临界 + 高潜力失常 | 填报结果与复读收益对比 | 情绪化复读、政策变化和心理成本 |
| 艺体扩展 + 就业落地 | 专业分/文化分双核验、城市资源、转向路径 | 只看综合分，不看未来就业 |
| 专项/定向扩展 + 体制路径 | 资格、义务、服务期和违约成本审计 | 把定向机会误认为无成本保障 |

## 不适合策略识别

正式建议中必须列出不适合采用的策略：

- 高分宽选择考生不适合只因热门城市或热门专业牺牲强学科和长期上限。
- 数学弱、抽象能力弱的学生不适合只因热门叙事进入高数学负担专业。
- 身心边界敏感学生不适合远距离、高压、强竞争和转专业豪赌叠加。
- 低预算家庭不适合把高收费项目作为保底，除非收益、退出和风险储备清晰。
- 本专科临界考生不适合无条件追求低质量本科身份。
- 目标不清晰学生不适合过早锁死窄行业、窄专业或强义务路径。
- 家长强控制且学生沉默时，不适合只按家长满意度排序。

## 报告输出要求

在正式报告的“考生画像摘要”中加入：

- 主画像类型：录取空间主判断。
- 次级约束画像：能力、家庭、身心、预算或规则约束。
- 长期路径倾向：升学、就业、体制、证照、行业、国际、家庭资源承接或探索保留。
- 不适合采用的策略：至少列出 1-3 条。
- 画像置信度：说明哪些信息已明确，哪些仍需补问或核验。
