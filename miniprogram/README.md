# Lever-GaoKao 微信小程序壳工程

这是一个轻量微信小程序入口，用于把 `Lever-GaoKao` 的问诊流程、使用边界和提示词复制能力交给低门槛用户。

## 功能

- 首页说明项目定位、适用场景和官方核验提醒。
- 问诊页收集省份、年份、选科/科类、分数、位次、批次、预算、不可接受项和家庭约束。
- 支持本地保存草稿，并一键复制给 Agent/老师/公益咨询志愿者继续分析的提示词。
- 使用边界页强调非商用、非录取保证、不得替代官方系统。

## 本地预览

1. 使用微信开发者工具导入本目录 `miniprogram/`。
2. AppID 可先使用游客模式或替换 `project.config.json` 中的 `appid`。
3. 预览前请结合正式主体资质、隐私政策和微信平台规范补齐发布材料。
4. 如需上线，请按仓库根目录的 [微信小程序部署指南](../docs/miniprogram-deployment.md) 完成 AppID 替换、上传、提审和发布。

## 注意

小程序当前不连接后端，也不上传用户数据。问诊草稿仅保存在用户本机微信小程序本地存储中。
